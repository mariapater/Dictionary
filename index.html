a

It is the element that allows us to link our HTML documents to each other, or to other websites external to ours. The most common and basic attribute for this tag is href, which allows us to specify where it will take us when we click.

adress

It shows the contact information of a person, such as name and email address, that is, with this tag, we can specify relevant contact information about the person, company that makes a website, but its use is not intended for a physical address of a place, but rather for the online contact details.

body

This element defines the main body of the HTML document that we are editing. In HTML 4 this tag had the possibility of specifying attributes, but they were deprecated, while in HTML 5 they have been definitively eliminated, so the tag no longer has any possible attributes.

button

With this tag, we can define buttons on the web. It is one of the most used elements in forms for various actions, such as submitting the form, accessing a directory tree, etc. In HTML 5, new attributes have been added to this element that facilitate its integration with the forms and allow us to tell the browser more easily the different actions that are carried out when the button is pressed.

caption

It is used to add a general title to a table without having to resort to the tag and styles, in such a way that a title can be added without corresponding to the format of the rest of the table.

details

They are defined in the same way as combos serves to display a list of values ​​on the web, the tag

div

It belongs to the set of elements that allow text highlighting. All content that goes into this tag will be considered the definition of a word. This is useful when creating a dictionary or glossary of terms.

em

Among the elements for text highlighting, it is the one that allows us to mark a text with greater importance, but from a semantic point of view and not from a visual point of view.

form

It is the tag used to define a form. In HTML 5, some new attributes have been added that bring new functionality to this element.

header

El tag header nos permite definir contenidos como preámbulo, que es muy útil en casos como una noticia, un documento de texto o un artítulo periodístico, así como una entrada de blog. Nos permite insertar texto que describa brevemente el contenido sobre el que va a tratar dicho documento, noticia o entrada.

html

It is the tag that defines the beginning and end of a document.

input

This element is used to define a field within a form. Various types of field can be defined with this tag, such as submit buttons, text fields, checkboxes, etc.

meta

It is a new HTML 5 tag that allows us to establish measures for a range that has a known maximum value, such as, for example, voting for different candidates, since there is a maximum number of possible votes that is known.

nav

It defines a part of the web as a navigation menu, that is to say, within this tag, a series of links is grouped, which is usually the menu of sections of the web, also known as the navigation menu.

alert

In JavaScript, an alert is a type of "pop-up" message to users of a script. An alert could be anything from notifying the user of an error to saying a short phrase such as "Hi".

algol

Formerly known as IAL, ALGOL is short for algorithmic language. It is a family of portable programming languages for scientific computations that had a strong influence on other languages.

algorithm

Derived from the name of the mathematician Muhammed ibn-Musa Al-Khowarizmi, an algorithm is a solution to a problem that meets the following criteria.

allocate

To allocate a resource is to designate or reserve it for a specific use. In computers, any limited resource may be allocated, such as RAM and disk space.

add

When referring to computers, add often refers to installing (adding) a hardware device or software program to the computer.

AOP

Short for aspect-oriented programming, AOP was originally developed by Xerox PARC and helps the programmer decide on how the program is going to be compiled.

API

Short for application programming interface, API is a set of routines, protocols, and tools for building software applications. APIs allow programmers easier entry into another company's program or service.

APK

Short for Android Package File, an APK is the file format which stores applications developed for the Android operating system.

Applet

Underline and strikethrough effects

Argument

Sometimes abbreviated as arg, referring to programming or the command line an argument is a value that is passed into a command, function, or routine. In the example below, "myfile.txt" is the argument for the edit command.

Array

When referring to programming, an array is a group of related data values (called elements) that are grouped together. All the array elements must be the same data type.

ASCII

Short for American Standard Code for Information Interexchange, ASCII is a standard that assigns letters, numbers, and other characters in the 256 slots available in the 8-bit code. The ASCII decimal (Dec) number is created from binary, which is the language of all computers.

ASP

Short for Active Server Page, an ASP page is a dynamically created web page that ends with a .ASP extension that utilizes ActiveX scripting, usually VBScript or JScript code.

ASPI

Short for advanced SCSI programming interface, ASPI is an interface specification for sending commands to a SCSI host adapter developed by Adaptec.
